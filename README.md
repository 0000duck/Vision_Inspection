# **Vision Inspection System**

## **Installation**

**Requirements**
- Basler USB3.0 VISION Camera
- PC with USB3.0 port
- Ubuntu(16.04+) (could extend to support Windows or other Linux)
- QT Library 5.11
- OpenCV-3.4.2
- Pylon 5.0.12 SDK

**Recommend IDE**
- QT Creator

**Steps:**

- Install OpenCV-3.4.2 from native building

- Install Pylon 5 SDK

- Install QT
